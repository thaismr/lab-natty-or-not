# Comandos de Instalação das Dependências

Execute os seguintes comandos no terminal para instalar as dependências necessárias:

```bash
npm install framer-motion
npm install embla-carousel-react
```

Ou, se preferir instalar ambas de uma vez:

```bash
npm install framer-motion embla-carousel-react
```

## Dependências Instaladas

- **framer-motion**: Biblioteca para animações suaves, especialmente para o mapa mental e flashcards
- **embla-carousel-react**: Biblioteca para criar o carousel de slides
