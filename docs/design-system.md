# Identidade Visual e UI/UX

## Temática e Estilo
O visual do projeto deve transmitir paz, clareza e um toque do sagrado. 
- **Estética:** Ilustrações em estilo aquarela e elementos com bordas suaves e arredondadas.
- **Cores:** Foco absoluto em tons pastéis.
  - Fundos: Bege muito claro, branco off-white, azul celestial pastel.
  - Textos: Cinza chumbo escuro ou marrom terroso suave (evite preto puro #000000 para reduzir o contraste agressivo).
  - Destaques (Botões/Ações): Dourado suave, terracota claro ou azul sereno.
- **Tipografia:** Fontes serifadas elegantes para títulos (remetendo a livros clássicos) e fontes sans-serif limpas para blocos de texto longo para facilitar a leitura.

## Bibliotecas Visuais Permitidas
- **Tailwind CSS:** Para toda a estrutura de grid, flexbox, tipografia e cores base.
- **Framer Motion:** Exclusivo para componentes React. Essencial para o efeito de "Shared Layout" ao expandir os nós do Mapa Mental, criando uma sensação de materialização suave do conteúdo.
- **Embla Carousel:** Biblioteca escolhida para a apresentação de slides. Deve ser implementada focando em performance, sem controles intrusivos.
